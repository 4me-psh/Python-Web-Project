# Python Web Project

This Python Web project is developed using Flask framework. It is constantly updated. The architecture used here is going to be an onion[1] architecture, with updates it will be improved.

Version 1.0.0 will be a starting point for the application. In this version the application got its basic appearance. This is a Python Web Project illustrating a Blog. 
Currently supported functions are: 
- register
- login/logout
- support of a session
basic operations with posts:
  - Not registered users can view posts.
  - Registered users can create posts, also update and delete their posts.
- categories, for now they only can be viewed by everyone and created by registered users. Further functionality will be added soon.

Tests will be added soon.

The database used is SQLite.
- [1] architecture choice might change in the future.
